The Maya project is currently in a start-up phase, as described in our revised five-year plan. The full information about the Maya project can be found in the Development Series. A quicker introduction, providing students with self-study material, can be found in the School Series.

We provide the following seven entry points, each one of which is roughly a factor ten more verbose than the previous one:

  • Title: The Maya Open Lab Project

  • Description: The Maya project aims at constructing an open laboratory for dense stellar systems. The presentation is centered around dialogues between two astrophysicists, Alice and Bob, who are developing the Kali code for simulating dense stellar systems. In the process, they construct an open lab, Maya, to set up initial conditions, run the Kali code, and analyze the results.
