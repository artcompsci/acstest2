El proyecto Kali se enfoca hacia la construcción de un laboratorio de un sistema estelar denso, como un cúmulo de estrellas, regiones de formación estelar, y nucleos galácticos que contienen uno o mas hoyos negros masivos. La presentación se muestra a través de dialogos entre dos astrofísicos quienes estan en proceso de la construcción de el código Kali desde el inicio.

Todavia es util, ofreciendo una explicación detallada del código Hermite de pasos de tiempo variable escrito en C++.
