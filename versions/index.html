ACS 1.0

Our first ACS version, ACS 1.0, was released on June 1, 2004.

It contains the versions, at that time, of the first four volumes of the Kali project, together with the first volume of the Open Knowledge umbrella project. In addition, the older volume Moving Stars Around, is included. In total, this release contains the equivalent of roughly 700 pages of printed text, together with the computer codes developed in the text.

We present ACS 1.0 in two option, each as a single tar file:

  • acs-1.0.tgf (xxx Mbyte), containing
    • all html files present on the ACS web site at the time this version was released
    • postscript and pdf versions of the text files
    • all computer codes discussed in the text
    • additional codes that are used for generating the .html, .ps, and .pdf files from the original .ok files

  • acs-lite-1.0.tgf (xx Mbyte), containing
    • a few top-level html files
    • all computer codes discussed in the text
    • the .ok files and codes that allow automatic construction of the .html, .ps, and .pdf files, with the exception of those connected with the MSA volume.
